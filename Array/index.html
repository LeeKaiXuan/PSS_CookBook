    Array

    Declarations

    Array has 2 types of declare syntaxes: Template, Square.

    1
2
3
4
    array<int    , 3> intArray   ;  ///<  declare integer array with 3 elements
array<bit [8], 3> byteArray  ;  ///<  declare byte array with 3 elements
array<string , 3> stringArray;  ///<  declare string array with 3 elements
array<array<int, 3>, 2> nestedArray;    ///<  declare nested array with 3x2 integer elements

    1
2
3
4
    int     intArray    [3];        ///<  declare integer array with 3 elements
bit [8] byteArray   [3];        ///<  declare byte array with 3 elements
string  stringArray [3];        ///<  declare string array with 3 elements
///< (1)!
    1. square not support for declare nested array

    Declare array by rand keyword:

    1
    rand array<int, 3> intArray;    ///<  declare integer array with 3 random elements

    1
    rand int intArray [3];          ///<  declare integer array with 3 random elements

    Array Operators

    Operator Description
    [] Used to access a specific element of an array.
    = Creates a copy of the array-type expression on the RHS and assigns it to the array on the LHS.
    == Evaluates to true if all elements with corresponding indexes are equal.
    != Evaluates to true if not all elements with corresponding indexes are equal.
    in It evaluates to true if the element specified on the left of the operator exists in the array collection on the right of the operator.
    foreach The foreach statement can be applied to an array to iterate over the array elements.

    Array Methods

    Method Description
    int size() Returns the number of elements in the arrays.
    int sum() Returns the sum of all elements currently stored in the array.
    list<data_type> to_list() Returns a list containing the elements of the array.
    set<data_type> to_set(): Returns a set containing the elements of the array.
